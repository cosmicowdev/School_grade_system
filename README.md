
# Sistema Escolar

Este projeto é um sistema escolar simples desenvolvido para fins de estudo de programação orientada a objetos e lógica de sistemas.  

O sistema permite o cadastro de alunos, professores e matérias, controle de login e gerenciamento de notas.  
A interface é feita em HTML, CSS e JavaScript, simulando a interação com o sistema em tela.

## Funcionalidades

- Login de professor e aluno
- Validação de usuários cadastrados
- Painel do professor para lançamento de notas
- Painel do aluno para consulta de média
- Controle de notas por aluno
- Interface gráfica no navegador

## Tecnologias utilizadas

- Java para estrutura lógica do sistema
- HTML para estrutura da interface
- CSS para estilização da interface
- JavaScript para interação das telas

## Estrutura do projeto

- Aluno.java  
- Professor.java  
- Materia.java  
- SistemaEscola.java  
- Main.java (versão terminal)  
- index.html (versão interface gráfica)

## Como executar a versão em terminal

1. Compile os arquivos Java  
   javac *.java  

2. Execute o programa  
   java Main  

3. Use o menu de login no terminal

## Como executar a interface gráfica

1. Abra o arquivo index.html em qualquer navegador  
2. Utilize os logins de teste cadastrados  
   Professor: P1 ou P2  
   Aluno: A1  

## Objetivo do projeto

Projeto criado para praticar:

- Programação orientada a objetos em Java
- Estruturas de dados
- Lógica de sistemas
- Criação de interface simples
- Integração entre front-end e lógica de aplicação
